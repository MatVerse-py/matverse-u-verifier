# MATVERSE GLOBAL PIPELINE

## Objective
Create a unified execution → proof → ledger → anchor pipeline across all repositories.

## Pipeline
1. Execute repo
2. Generate event
3. Hash event (SHA256)
4. Append to global ledger
5. Generate Merkle root
6. Anchor (Sepolia / Zenodo)

See orchestrator in Cassandra output.
